25th Jun 2023

Git + GitHub + Gradescope Workshop

CS Club presents Git Gud with Git, Github and Gradescope 🔥
Now that the university is moving to Git, Github and Gradescope we would like to invite all our peers, first years or otherwise to this in-person and online workshop.
We will be covering the basics of Git and Github, what they are and how to use Gradescope alongside these tools during your courses. We will also go through how Gradescope works when using the Code50 IDE.
We will not be covering SVN during the session as first year courses are now using Github and second years and up should already be familiar with SVN.
Hope to see you all there so we can Git Gud together!
